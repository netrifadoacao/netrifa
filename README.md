Documentação do sistema de renda recorrente universal
VERSÃO: 1.0
TIPO: sistema web com escritório virtual e painel administrativo.
Pagamentos: integração com mercado pago.
visão geral do sistema 
o sistema de vendas é uma plataforma digital que permite a venda, distribuição e gestão de produtos e serviços físicos e digitais por meio de um modelo de rede, (multinível) onde cada comprador automaticamente se torna um vendedor, através de um escritório virtual próprio.
O sistema possui duas áreas principais.
Escritório virtual (usuários/vendedores/compradores
Área administrativa (administrador do sistema

O processo é totalmente automatizado, desde o cadastro até o pagamento e distribuição de bônus.

fluxo geral de funcionamento
O usuário acessa um link de indicação de um participante.
Ao se cadastrar ele poderá comprar e vender produtos e serviços digitais
Realiza o pagamento via mercado pago no próprio link.

3 Esse cadastro gera um escritório virtual.
4 O novo usuário passa a: comprar e vender produtos físicos e digitais por meio do seu próprio link de cadastro.
5 As vendas geram bônus diretos e em rede, conforme configuração do administrador.

escritório virtual do usuário
O escritório virtual é o painel onde compradores e vendedores gerenciam suas atividades.
